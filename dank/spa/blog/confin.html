CONFIN
Dec. 18, 2017

CONFIN: programa de apoyo al control financiero.


Este programa MOFIN, diseñado por Ramón E. García, es un sistema contable para organizaciones de base, cooperativas y asociaciones. Este programa permite incluir datos y procesar cuadros de salida como el estado financiero y el balance general. Este programa puede ser adaptado para cualquier tipo y tamaño de organización.

La innovación está en el uso del programa, en que los órganos de una organización lo usen para sus deliberaciones y decisiones. ¿Cómo hacerlo? Dichos órganos van a usar esos datos si aumenta su capacidad de análisis de datos (cuadros de salida) y si el área contable les provee esa información inmediatamente a su solicitud. Ambos puntos, flujo de información y análisis de los datos, son posibles si COSERPROSS les acompaña al menos por un período de tiempo.
