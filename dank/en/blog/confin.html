CONFIN: financial control support program


This program, MOFIN, was designed by Ramón E. García. It is an accountant system for base organizations, cooperatives, and associations. This program allows the inclusion of data and the processing of frames of exit like the financial status and balance in general. This program can be adapted to any type and size of organization.

The innovation is with the use of this program, organs of an organization are able to utilize it for their deliberations and decision. How to do it? Said organs will use these datasets if their data analysis capabilities increase (frames of exit) y if the accounting area provides them of this information immediately after request. Both points, flux of information and dataset analysis, are possible if COSERPROSS walks them through it for at least some period of time.
